# Cattle Treatment Tracker

A multi-herd cattle treatment tracking web application built with Python Flask. Features an integrated LA-200 dosage calculator for quick and accurate medication administration.

## Features

- **Multi-Herd Management** - Track treatments separately for different pastures/herds
- **LA-200 Dosage Calculator** - Weight-based dosing chart (100-1200 lb cattle)
- **Multiple Medications** - Record multiple treatments per animal
- **Treatment History** - View, edit, and delete past treatments
- **Smartphone Interface** - Optimized mobile-friendly design
- **Persistent Storage** - JSON-based data storage
- **Timestamps** - Automatic date/time tracking for all treatments

## Quick Start

### Prerequisites

- Python 3.6+
- Flask 3.0.0

### Installation

1. Clone this repository:
```bash
   git clone https://github.com/boydhanagriff21/cattle-treatment-app.git
   cd cattle-treatment-app
```

2. Install dependencies:
```bash
   pip3 install -r requirements.txt
```

3. Run the application:
```bash
   python3 app.py
```

4. Open your browser and navigate to:
```
   http://localhost:5001
```

## Usage

1. **Select Your Herd** - Choose existing or create new herd/pasture
2. **Add Treatment** - Enter tag number, select medication, add notes
3. **LA-200 Calculator** - Click weight range for automatic dosing
4. **Multiple Medications** - Check box to add more medications per animal
5. **View Records** - See all treatments with edit/delete options
